Віджэт нататак выкарыстоўвае заснаваны на [Markdown](https://www.markdownguide.org/cheat-sheet/) сінтаксіс, які дазваляе ствараць загалоўкі, спісы і палі для птушак. Вы можаце дадаць наступныя сімвалы ў пачатак радка:

- `#` асноўны загаловак
- `##` падзагаловак
- `###` падпадзагаловак
- `-` калсічны спіс
- `[ ]` поле для птушкі (звярніце ўвагу на прабел паміж дужкамі)
- `[x]` адзначанае поле для птушкі

<LazyVideo src="/documentation/overview/notes.webm" />

Іншыя асаблівасці Markdown (напрыклад, курсіўны тэкст) у цяперашні момант недаступныя.

### Спалучэнні клавіш

Падчас рэдагавання нататак можна выкарыстоўваць наступныя спалучэнні:

| Клавішы                                                            | Дзеянне                    |
| ------------------------------------------------------------------ | -------------------------- |
| <kbd>Ctrl</kbd> / <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>1</kbd> | Стварыць асноўны загаловак |
| <kbd>Ctrl</kbd> / <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>2</kbd> | Стварыць падзагаловак      |
| <kbd>Ctrl</kbd> / <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>3</kbd> | Стварыць падпадзагаловак   |
| <kbd>Ctrl</kbd> / <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>4</kbd> | Стварыць звычайны спіс     |
| <kbd>Ctrl</kbd> / <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>5</kbd> | Стварыць спіс спраў        |
