Віджэт цытат можа сінхранізаваць цытаты з розных крыніц:

- Прадвызначана — з нашага **[адкрытага шматмоўнага API з цытатамі](https://github.com/victrme/i18n-quotes)**. Тут сабраныя матывацыйныя або натхняльныя цытаты, якімі суполка можа кіраваць праз запыты на дадаванне.
- API **[InspiroBot](https://inspirobot.me/)** — «олд-скульны» ШІ, які генеруе на ангельскай мове быццам матывацыйныя цытаты, але з дасціпным падколам. Ён з'яўляецца пародыяй традыцыйных матывацыйных цытат, але з гледзішча ШІ. Хаця мы ніякім чынам не звязаныя з InspiroBot, нам гэты сэрвіс заўжды падаваўся забаўным, і мы падумалі, што было б добра дадаць яго ў Bonjourr. Трэба заўважыць, што ён створаны прынамсі 10 гадоў таму і <u>**аніяк не звязаны з сучаснымі ШІ-тэндэнцыямі**</u>.
- API **[Kaamelott](https://github.com/sin0light/api-kaamelott)** — цытаты з францускай тэлекамедыі. Даступныя толькі па-француску.
- **Стаічныя (Stoic)** цытаты, якія адлюстроўваюць філасофію [стаіцызму](https://be.wikipedia.org/wiki/Стаіцызм). Даступныя толькі па-ангельску.
- **Хітокото (Hitokoto)** — японскія натхняльныя цытаты. Даступныя толькі па-японску.
- **Уласныя** — дадавайце свае ўласныя цытаты наўпрост праз інтэрфейс Bonjourr.
- **URL** — дадавайце цытаты з аддаленага CSV-файла.

### Імпарт уласных цытат

Свае цытаты можна дадаць двума спосабамі: наўпрост праз Bonjourr або з аддаленага URL. У абодвух выпадках варта прытрымлівацца наступнага фарматавання:

<Code
	code={
		"Аўтар, Цытата.\nАўтар, Яшчэ адна цытата.\\nДы яшчэ з пераносам на наступны радок!"
	}
	lang="json"
/>

#### Наўпрост праз інтэрфейс

Пасля выбару тыпу цытат «Уласныя» вы можаце проста дадаваць цытаты, выконваючы паказаныя вышэй правілы фарматавання.

Пры такім варыянце тэкст абмежаваны да 8 кб, што раўняецца прыкладна 50 тыповым цытатам. Прычына таму — API [storage.sync](https://developer.mozilla.org/en-US/docs/Mozilla/Add-ons/WebExtensions/API/storage/sync), праз які Bonjourr захоўвае даныя і які абмяжоўвае памер кожнага аб'екта да 8 кб. **Калі вам патрабуецца больш цытат, раім вам выкарыстоўваць аддалены URL.**

#### З аддаленага URL-адрасу CSV-файла

Каб прывязаць Bonjourr да прамога URL-адрасу CSV-файла з усімі вашымі цытатамі, выберыце варыянт «Аддалены URL».

Найпрасцей гэта можна зрабіць праз GitHub Gist — сэрвіс, які дазваляе вельмі лёгка хосціць файлы:

1. Стварыце ўліковы запіс на [GitHub](https://github.com), калі ў вас яго няма, і ўвайдзіце ў яго.
2. Перайдзіце на старонку [GitHub Gist](https://gist.github.com/).
3. Дайце файлу апісанне — напрыклад, «Мае цытаты для Bonjourr».
4. Назавіце файл — напрыклад, `bonjourr_quotes.csv` ці `cytaty_bonjourr.csv`. Назва можа быць любая, але павінна скончвацца на `.csv`.
5. Пачніце ўводзіць цытаты, прытрымліваючыся фарматавання, паказанага вышэй (калі вы захочаце, вы заўжды зможаце іх адрэдагаваць).
6. Калі скончыце, націсніце **Create secret gist**. Файл захаваецца і будзе адлюстраваны ў выглядзе табліцы.
7. Націсніце <kbd>Raw</kbd>, скапіюйце URL-адрас старонкі, якая пасля адкрыецца, і ўстаўце яго ў адпаведнае поле ў Bonjourr. Вуаля!
