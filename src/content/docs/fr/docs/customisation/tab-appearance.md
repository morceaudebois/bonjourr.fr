---
title: Apparence de l’onglet
description: L’apparence de l’onglet permet de personnaliser le titre et le favicon de Bonjourr dans le navigateur, à l’aide de texte et d’emojis, avec certaines limites selon les navigateurs.
---

Il est possible de personnaliser le texte et l’icône (favicon) affichés par le navigateur pour Bonjourr. Le texte est limité à 80 caractères, et les icônes peuvent uniquement être des emojis.

<img
  src="/documentation/overview/tab-appearance.png"

L’option emoji peut être limitée selon le navigateur utilisé :

- Firefox peut présenter des problèmes d’alignement ou de taille ([problème GitHub](https://github.com/victrme/Bonjourr/issues/431)).
- Microsoft Edge et Safari ne la prennent pas en charge.

Voici comment saisir des emojis :

- Windows : <kbd>WIN</kbd> + <kbd>.</kbd>
- macOS : <kbd>Ctrl</kbd> + <kbd>Cmd</kbd> + <kbd>Space bar</kbd>
- Il est aussi possible de les copier-coller depuis Internet : [unicode.org/emoji/charts/full-emoji-list](https://unicode.org/emoji/charts/full-emoji-list.html).
