## 📄 Mise en page

Trois types de mise en page sont disponibles : une, deux ou trois colonnes. Chaque configuration conserve indépendamment la position et l’alignement des widgets.

Ces options n’affectent que la mise en page active :

- Activation ou désactivation des widgets
- Tous les réglages du module de grille
- Réinitialisation de la mise en page

## 📚 Polices

Bonjourr utilise les polices proposées par Fontsource. Pour visualiser l’ensemble des typographies disponibles, il est possible de consulter [fontsource.org](https://fontsource.org/).
