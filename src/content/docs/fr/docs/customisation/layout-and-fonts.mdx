## 📄 Mises en page {#page-layouts}

Bonjourr propose trois mises en page : une, deux ou trois colonnes. Chaque mise en page possède ses propres positions de widgets et réglages d'alignement, ce qui vous permet de créer différentes configurations et de passer de l'une à l'autre.

Les paramètres suivants sont enregistrés séparément pour chaque mise en page :

- Activation des widgets (interrupteurs)
- Toutes les options de l'outil de grille
- Réinitialiser la mise en page

## 📚 Polices {#fonts}

Bonjourr utilise les polices de Fontsource, un catalogue de polices **open source**. Vous pouvez choisir parmi des centaines de polices pour personnaliser l'apparence de votre nouvel onglet.

Pour prévisualiser les polices disponibles, rendez-vous sur [fontsource.org](https://fontsource.org/).

[![Prévisualisez les polices de Bonjourr depuis Fontsource](@assets/documentation/customisation/fontsource.png)](https://fontsource.org/)

### Et les autres polices ? {#other-custom-fonts}

Utilisez du CSS dans le champ **Style personnalisé** pour appliquer à Bonjourr [n'importe quelle police installée sur votre ordinateur](https://wordmark.it/). Par exemple :

```css
/* applique une police personnalisée à tous les widgets */
#interface {
    font-family: Comic Sans MS;
}
```

Vous pouvez même utiliser les propriétés `@import` ou `@font-face` pour importer des polices depuis une URL distante provenant de [Google Fonts](https://fonts.google.com) ou de tout autre service de polices.

```css
/* applique une police Google Fonts à tous les widgets */
@import url('https://fonts.googleapis.com/css2?family=IM+Fell+English:ital@0;1&display=swap');

#interface, #pomodoro_container {
  font-family: "IM Fell English", serif;
  font-weight: 400;
  font-style: normal;
}
```
