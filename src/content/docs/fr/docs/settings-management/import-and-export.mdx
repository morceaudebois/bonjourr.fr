## Comment exporter et importer mes paramètres ?

Bonjourr vous permet d'**exporter** tous vos paramètres dans un fichier. Vous pouvez utiliser cette fonctionnalité pour les sauvegarder ou les partager avec quelqu'un. Vous pourrez ensuite **importer** ces paramètres dans n'importe quelle instance de Bonjourr.

Pour cela, rendez-vous simplement en bas du panneau des paramètres, puis cliquez sur le bouton <kbd>Export</kbd> dans la section de gestion des paramètres. Votre fichier sera téléchargé. Sur votre autre instance de Bonjourr, cliquez sur le bouton <kbd>Import</kbd> et sélectionnez le fichier précédemment exporté. Bonjourr se rechargera et tous vos paramètres seront appliqués.

<Aside type="caution">
	Le fichier de paramètres contient **tout ce que vous avez déjà modifié dans Bonjourr**, à l'exception des fichiers locaux
	que vous avez importés (arrière-plans et icônes personnalisés).
</Aside>

Vous pouvez le modifier manuellement avec n'importe quel éditeur de texte, et vous trouverez [ici](/fr/documentation/reference/settings-reference/) les correspondances entre chaque section et son contenu.

## Comment réinitialiser Bonjourr ?

Vous trouverez le bouton <kbd>Reset</kbd> tout en bas du panneau des paramètres de Bonjourr. Appuyez dessus deux fois et tous vos paramètres seront supprimés, réinitialisant l'extension à son état d'origine.
