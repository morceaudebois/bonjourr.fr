## Comment exporter et importer mes paramètres ?

Bonjourr vous permet d'exporter tous vos paramètres dans un fichier. Vous pouvez ainsi les sauvegarder ou les partager avec quelqu'un, puis les importer dans n'importe quelle autre instance de Bonjourr.

Pour cela, rendez-vous tout en bas du panneau des paramètres, puis cliquez sur le bouton <kbd>Exporter</kbd> dans la section de gestion des paramètres. Le fichier sera téléchargé. Sur l'autre instance de Bonjourr, cliquez sur <kbd>Importer</kbd>, sélectionnez le fichier précédemment exporté, puis Bonjourr rechargera automatiquement la page et appliquera tous vos paramètres.

<Aside type="caution">
	Le fichier de paramètres contient toutes les modifications que vous avez effectuées dans
	Bonjourr, à l'exception des fichiers locaux que vous avez importés (arrière-plans et icônes
	personnalisés).
</Aside>

Vous pouvez également le modifier avec n'importe quel éditeur de texte. Si besoin, vous trouverez la description de chaque section [ici](/fr/docs/reference/settings-reference/).

## Comment réinitialiser Bonjourr ?

Le bouton <kbd>Réinitialiser</kbd> se trouve tout en bas du panneau des paramètres de Bonjourr. Cliquez dessus deux fois pour supprimer tous vos paramètres et restaurer l'extension à son état d'origine.
