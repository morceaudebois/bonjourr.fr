## Types de citations

Le widget Citations peut récupérer des citations depuis plusieurs sources grâce à l'option __Type de citations__ :

- Par défaut, notre **[API de citations open source](https://github.com/victrme/i18n-quotes)** disponible dans plusieurs langues. Ce sont des citations générales motivantes ou inspirantes que la communauté peut gérer via des pull requests.
- L'**API [InspiroBot](https://inspirobot.me/)**, une ancienne IA qui génère des citations apparemment inspirantes, mais avec une touche humoristique. Elle parodie les citations de motivation traditionnelles du point de vue d'une IA. Bien que nous ne soyons pas affiliés à InspiroBot, nous avons toujours trouvé ce service amusant et avons pensé qu'il serait intéressant de l'intégrer à Bonjourr. Il est important de noter qu'elle a au moins 10 ans et <u>**n'a rien à voir avec la tendance récente de l'IA générative**</u>.
- L'**API [Kaamelott](https://github.com/sin0light/api-kaamelott)**, des citations issues d'une série télévisée humoristique française. Disponible uniquement en français.
- Des citations **Stoïciennes** qui reflètent la [philosophie du stoïcisme](https://en.wikipedia.org/wiki/Stoicism).
- **Hitokoto**, des citations inspirantes japonaises.
- **Personnalisé**, ajoutez vos propres citations directement depuis l'interface de Bonjourr.
- **URL**, ajoutez des citations depuis un fichier CSV distant.

<Aside type="tip">Faites un clic droit sur le widget Citations pour passer à la suivante.</Aside>

#### Avec l'option citations personnalisées

Une fois l'option citations personnalisées sélectionnée, vous pouvez simplement ajouter des citations en respectant le format indiqué ci-dessus.

Cette option est limitée à 8 ko de texte, soit environ 50 citations de taille standard. Cela est dû au fait que Bonjourr utilise [storage.sync](https://developer.mozilla.org/en-US/docs/Mozilla/Add-ons/WebExtensions/API/storage/sync) pour stocker les données, et que chaque objet est limité à 8 ko. **Si vous avez besoin de plus de citations, nous vous conseillons plutôt d'utiliser l'option URL.**

#### Avec l'option URL (fichier CSV)

Sélectionnez l'option URL pour relier Bonjourr à l'URL directe d'un fichier CSV contenant toutes vos citations.

La méthode la plus simple consiste à utiliser GitHub Gist, un service qui permet de créer et d'héberger très facilement un fichier :

<Steps>
1. #### Créer un compte GitHub

   Créez un compte [sur GitHub](https://github.com) si vous n'en avez pas déjà un, puis connectez-vous.

2. #### Créer un GitHub Gist

   Rendez-vous sur la [page GitHub Gist](https://gist.github.com/).

   Donnez une description à votre fichier, comme `Mes citations Bonjourr`, puis choisissez un nom de fichier comme `bonjourr_quotes.csv`. Le nom peut être n'importe lequel tant qu'il se termine par `.csv`.

3. #### Ajouter vos citations

   Commencez à écrire vos citations en utilisant le formatage indiqué ci-dessus. Vous pourrez les modifier plus tard si vous le souhaitez.

4. #### Créer le Gist

   Une fois terminé, cliquez sur le bouton **Create secret gist**. Le fichier sera enregistré et affiché sous forme de tableau.

5. #### Copier l'URL brute

   Cliquez sur le bouton <kbd>Raw</kbd>, puis copiez l'URL de la page qui s'ouvre.

6. #### L'ajouter dans Bonjourr

   Collez l'URL dans le champ "URL" de Bonjourr. Voilà !
</Steps>
