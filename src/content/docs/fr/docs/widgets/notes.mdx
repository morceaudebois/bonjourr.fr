## Utiliser le formatage des notes

Notes utilise une syntaxe similaire au [Markdown](https://www.markdownguide.org/cheat-sheet/) pour créer des titres, des listes et des cases à cocher. Pour formater une ligne, ajoutez simplement ces caractères au début de celle-ci :

- `#` grand titre
- `##` titre moyen
- `###` petit titre
- `-` liste à puces
- `[ ]` case à cocher (notez l'espace entre les crochets)
- `[x]` case à cocher activée

<LazyVideo src="/documentation/widgets/notes.mp4" />

**D'autres fonctionnalités Markdown, comme le gras, l'italique et les liens, ne sont actuellement pas prises en charge.**

### Raccourcis clavier

Vous pouvez utiliser ces raccourcis clavier lorsque vous modifiez votre note.

| Touches                                                            | Action                    |
| ------------------------------------------------------------------ | ------------------------- |
| <kbd>Ctrl</kbd> / <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>1</kbd> | Créer un grand titre      |
| <kbd>Ctrl</kbd> / <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>2</kbd> | Créer un titre moyen      |
| <kbd>Ctrl</kbd> / <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>3</kbd> | Créer un petit titre      |
| <kbd>Ctrl</kbd> / <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>4</kbd> | Créer une liste à puces   |
| <kbd>Ctrl</kbd> / <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>5</kbd> | Créer une liste de tâches |
