## Utilisation générale

Faites un clic droit sur le widget horloge et date pour ouvrir ses réglages. Vous pourrez ensuite modifier les fuseaux horaires, le format de date, la taille et d'autres options d'affichage.

## Horloge numérique

L'horloge numérique par défaut inclut des options de configuration classiques comme le format 12/24 heures, l'affichage des secondes et les indicateurs AM/PM.

## Horloge analogue

Si vous souhaitez quelque chose de plus élégant, Bonjourr intègre une option d'horloge analogue très personnalisable. Forme, style du cadran, aiguilles, arrière-plan, bordure... Vous pouvez modifier de nombreux éléments pour créer l'horloge qui vous correspond.

![Les styles d'horloge de Bonjourr](@assets/documentation/widgets/clock-styles.png)

Quelques points utiles à connaître lors de la personnalisation :

- Cliquer sur les icônes 🌙/☀️ des curseurs permet de passer du noir au blanc.
- L'arrière-plan de l'horloge analogue est flouté lorsque son opacité est supérieure à 0.
- Activer les secondes sur l'horloge analogue est généralement recommandé.

## Horloges mondiales

![La fonctionnalité d'horloges mondiales de Bonjourr](@assets/documentation/widgets/world-clocks.png)

Les **horloges mondiales** permettent d'ajouter plusieurs horloges avec différents fuseaux horaires. Supprimer le nom d'une horloge mondiale supprimera entièrement cette horloge.
