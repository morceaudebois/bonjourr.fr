La géolocalisation précise nécessite l’accès à la position de votre appareil. Certains navigateurs ou systèmes peuvent bloquer cette autorisation selon les réglages de confidentialité. macOS / Safari pose parfois problème à ce niveau.

Si plusieurs villes portent le même nom dans un pays, cela peut entraîner des erreurs. Dans ce cas, il est conseillé de choisir la ville la plus proche avec un nom unique.

Bonjourr ne propose pas encore de bouton de rafraîchissement manuel pour la météo. Il est toutefois possible de forcer une mise à jour en alternant les unités (Celsius → Fahrenheit → Celsius).

### Météo détaillée

La **météo détaillée** permet d’accéder rapidement à un site météo externe pour consulter un rapport plus complet. Une fois activée, un clic sur le widget météo ouvre directement le service choisi.
