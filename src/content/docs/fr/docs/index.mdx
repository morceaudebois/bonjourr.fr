## Autres navigateurs de bureau

### ✅ Brave

Étant basé sur Chromium, Brave permet d'installer n'importe quelle extension Chrome. Vous pouvez [télécharger Bonjourr](https://chrome.google.com/webstore/detail/bonjourr-%C2%B7-minimalist-lig/dlnejlppicbjfcfcedcflplfjajinajd) et commencer à l'utiliser immédiatement.

### ✅ Safari (macOS)

Bonjourr peut être installé sur Safari pour macOS en quelques secondes :
 
<Steps>
1. #### Ouvrez les préférences de Safari
    Appuyez sur <kbd>Cmd</kbd> + <kbd>Virgule</kbd> et ouvrez l'onglet « Général »
2. #### Passez à la page d'accueil
    Définissez « Les nouvelles fenêtres s'ouvrent avec » et « Les nouveaux onglets s'ouvrent avec » sur « Page d'accueil ».
3. #### Définissez votre page d'accueil
    Définissez le champ Page d'accueil sur : <Code code={"https://online.bonjourr.fr"} />
</Steps>

### ✅ Zen

Zen est basé sur Firefox, ce qui signifie que vous pouvez y installer n'importe quelle extension Firefox, y compris [Bonjourr](https://addons.mozilla.org/firefox/addon/bonjourr-startpage/). Zen nécessite quelques réglages supplémentaires avant que Bonjourr ne remplace complètement les nouveaux onglets.

<Steps>
1. #### Définissez Bonjourr comme page d'accueil et nouvel onglet

   Ouvrez les paramètres de Zen et assurez-vous que Bonjourr est sélectionné pour « Page d'accueil et nouvelles fenêtres » ainsi que « Nouveaux onglets ».

2. #### Désactivez le nouvel onglet personnalisé de Zen

   Ouvrez `about:config`, cliquez sur **Accepter le risque**, puis recherchez `zen.urlbar.replace-newtab`. Définissez sa valeur sur `false` à l'aide du bouton bascule à droite. Cela garantit que Bonjourr s'affiche à chaque ouverture d'un nouvel onglet.

3. #### (Optionnel) Ajoutez Bonjourr aux éléments essentiels

   Ouvrez un onglet Bonjourr, puis cliquez sur **Ajouter aux éléments essentiels**. Cela permet de garder Bonjourr facilement accessible depuis la barre latérale de Zen.
</Steps>

### ✅ Vivaldi

La prise en charge des extensions de page de nouvel onglet par Vivaldi s'est nettement améliorée dans les dernières versions. Une fois Bonjourr installé depuis le [Chrome Web Store](https://chrome.google.com/webstore/detail/bonjourr-%C2%B7-minimalist-lig/dlnejlppicbjfcfcedcflplfjajinajd), vous devrez modifier deux paramètres :

<Steps>
1. #### Onglet Général
    Cherchez la section « Au démarrage » et cochez la case « Pages spécifiques ». Copiez ensuite ceci dans le champ des pages de démarrage avant de cliquer sur <kbd>Ajouter</kbd> :
    <Code code={"chrome-extension://dlnejlppicbjfcfcedcflplfjajinajd/index.html"} lang="json" />
    ![Onglet des paramètres généraux de Vivaldi](@assets/documentation/getting-started/vivaldi-general.jpg)
2. #### Onglet Onglets
    Assurez-vous que la page de nouvel onglet peut être contrôlée par une extension : ![Onglet des paramètres des onglets de Vivaldi](@assets/documentation/getting-started/vivaldi-tabs.jpg)

### 🟡 Opera (et Opera GX)

Opera n'autorise pas les extensions à modifier son nouvel onglet. Bien qu'il n'y ait pas de déclaration officielle, nous pensons que cela est dû au fait que les publicités sur leur Speed Dial constituent une source de revenus importante pour eux.

Une solution consiste à installer une extension et à utiliser la version en ligne de Bonjourr :

<Steps>
  1. Videz votre Speed Dial actuel afin qu'il affiche une page vide sombre.

  2. Vous devez ensuite installer une extension tierce capable de remplacer votre onglet actif actuel. Pour le moment, nous avons constaté que seule [New Tab Start Page](https://chromewebstore.google.com/detail/new-tab-start-page/mjfahldkpjhcnfbbmdhpjolcjpcfhcpj) fonctionne correctement. Nous ne sommes pas affiliés à cette extension.

  3. Cliquez sur l'icône de Bonjourr, son URL complète devrait apparaître dans la barre d'adresse.

  4. Configurez l'extension tierce avec `https://online.bonjourr.fr`, et c'est terminé !
</Steps>

### 🟡 Orion

## Navigateurs mobiles

La prise en charge des extensions de nouvel onglet n'est pas idéale sur mobile, mais il existe tout de même plusieurs façons d'utiliser Bonjourr sur téléphones et tablettes (et même les réfrigérateurs connectés).

### 🟡 Safari (iOS et iPadOS)

Sur iOS et iPadOS, Apple autorise uniquement les applications de l'App Store à modifier la page de nouvel onglet de Safari. Même si [nous n'en avons plus](https://ko-fi.com/post/On-why-we-wont-be-renewing-our-Apple-Developer-ac-W7W31WQTJ5), il existe toujours une façon d'utiliser Bonjourr sur Safari grâce à l'application open source [Homepage for Safari](https://apps.apple.com/app/homepage-for-safari/id6481118559).

Cette application permet simplement de définir n'importe quelle URL comme page de nouvel onglet dans Safari. En l'utilisant pour définir `https://online.bonjourr.fr`, vous obtiendrez Bonjourr.

### 🟡 Navigateurs autres que Safari sur iOS

Il n'existe aucun moyen pour les navigateurs autres que Safari sur iOS d'installer des extensions.

Certains d'entre eux (comme [Firefox](https://apps.apple.com/app/navigateur-firefox-rapide-s%C3%BBr/id989804926)) permettent cependant de définir une URL personnalisée à l'ouverture de nouveaux onglets, dans laquelle vous pouvez mettre `https://online.bonjourr.fr`. Même s'il s'agit de la version en ligne de Bonjourr, ses données seront stockées dans le cache, ce qui signifie qu'elle restera très rapide.

### ❌ Chrome pour Android

La version Android de Chrome ne possède pas de système d'extensions, ce qui rend l'installation de Bonjourr impossible.

La seule façon d'installer Bonjourr sur Android était d'utiliser Kiwi Browser, qui permettait d'installer des extensions Chrome classiques. **Cela n'est plus possible depuis l'arrêt du développement de Kiwi.**

<Aside type="note">Certains [développements récents](https://www.reddit.com/r/kiwibrowser/comments/1i89rje/kiwi_browser_extension_code_integrated_into/) indiquent que le code source de Kiwi a été intégré à [Microsoft Edge pour Android](https://play.google.com/store/apps/details?id=com.microsoft.emmx). Vous pourriez peut-être y installer Bonjourr, même si nous ne l'avons pas testé.</Aside>

### ❌ Firefox pour Android

Mozilla empêche les extensions de modifier la page d'accueil sur sa version Android, ce qui rend l'installation de Bonjourr impossible. Tant qu'ils ne changeront pas cela, nous ne pouvons rien faire de notre côté.

Une autre possibilité serait qu'ils implémentent une option d'URL personnalisée. Cela permettrait d'utiliser la version en ligne de Bonjourr. Vous pouvez [voter pour cette proposition](https://connect.mozilla.org/t5/ideas/add-option-to-set-custom-homepage-in-firefox-for-android/idi-p/83481) sur leur forum.
