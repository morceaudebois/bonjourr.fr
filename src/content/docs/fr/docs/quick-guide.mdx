<LazyVideo src="/documentation/quick-guide/change-bg.mp4" controls />

## Comment ajouter une barre de recherche ?

<LazyVideo src="/documentation/quick-guide/add-search.mp4" controls />

## Comment ajouter une liste de tâches ?

Utilisez le [markdown](/fr/docs/widgets/notes/) dans le widget Notes pour créer des checklists :

<LazyVideo src="/documentation/quick-guide/todo.mp4" controls />
