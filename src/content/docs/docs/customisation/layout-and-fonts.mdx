## 📄 Page layouts {#page-layouts}

Bonjourr offers three page layouts: single, double, and triple column. Each layout has its own widget positions and alignment settings, allowing you to create different setups and switch between them.

The following settings are saved separately for each layout:

- Widget toggle (enable switches)
- All options in grid toolbox
- Reset layout

## 📚 Fonts {#fonts}

Bonjourr uses fonts from Fontsource, an **open-source** font catalog. You can choose from hundreds of typefaces to customise the look and feel of your new tab page.

To preview the available fonts, visit [fontsource.org](https://fontsource.org/).

[![Preview Bonjourr's fonts from Fontsource](@assets/documentation/customisation/fontsource.png)](https://fontsource.org/)

### But what about other fonts?

Use CSS in the **Custom Style** field and apply to Bonjourr [any font that's installed on your computer](https://wordmark.it/). For instance:
```css
/* sets a custom font on all widgets */
#interface {
    font-family: Comic Sans MS;
}
```

<Aside type="tip">You can even use the `@font-face` property to import a font from a remote URL coming from Google Fonts or any other font service. [Learn more](https://css-tricks.com/almanac/rules/f/font-face/)</Aside>
