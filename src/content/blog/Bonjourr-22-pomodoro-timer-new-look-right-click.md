Good news, everyone! This year, we'll be celebrating the holiday season with a brand new Bonjourr update. This one is... substantial, more so than we had previously planned. Here goes!

🍅 Introducing the Pomodoro timer

The Pomodoro timer is our first new widget in quite a long time, we're really excited about it! If you've never heard of the Pomodoro technique before, here's the short version: it's a time management method where you break your work into multiple intervals, for example: start working for 25 minutes, then take a 5-minute break, work again for 25 minutes until the timer rings, take a longer break and repeat.

This structured approach can help you stay motivated, avoid distractions and overall get more done in less time.

Our brand new Pomodoro in Focus mode

Bonjourr's Pomodoro timer includes three customizable intervals and a focus mode to hide everything else in Bonjourr while you're working, thus reducing the chances of you getting distracted. The Pomodoro will keep running even if you close all Bonjourr tabs and will ring at the end as long as you have at least one tab open. The whole thing is wrapped in a nice little UI inspired by Apple's Liquid Glass, and will even show you some stats if you right-click it.

🖱️ Universal right-click: a huge quality of life boost

What if I told you that you could right-click on anything in Bonjourr to get some contextual actions? Well in Bonjourr 22, you can! Want to edit your time settings? Right-click on the time! Skip to the next background? Right-click on the background! Want to create a new Quick Link? Well... right-click anywhere. 🙃

Easily access widget settings

This feature in itself makes Bonjourr 22 our most important update in recent years, as it will impact pretty much every single one of our dear users, making navigation faster and more intuitive... especially with how many settings sections we're starting to have. It also makes creating Quick Links so much easier.

Uses for the right click menu are plentiful

🎥 Import videos, video sound & import backgrounds through URLs

The Local Files background option now allows you to import videos as backgrounds, making it easier to mix things up: any photo or video you have on your computer can now be set as a background in Bonjourr. You can even play video sound now (although they'll be muted by default).

Import your own videos (now with sound!)Additionally, a new background type makes its appearance: URLs. Contrary to the Local files, this one makes it possible to fetch images from the Internet; simply set a direct link to an image and Bonjourr will fetch it. This may feel redundant, but it has two distinct perks:

    If your instance of Bonjourr lives in a particularly closed environment (like work or school), it will make it possible to set any image you want in the extension, as long as it's on the Internet. You won't need to have files on the computer anymore.
    Contrary to local files, URLs are synced to your other instances of Bonjourr.

🎨 Import link icons & round(able) links

In Bonjourr 22, gone is the confusing text input to manage link icons. It's replaced with a dropdown defaulting to "Website favicon", the regular favicon fetching option that works the same as it always has.

Easy custom icons from local files

But this dropdown gives you two more options:

    Local file, which (finally!) makes it possible to import any image from your computer as a link icon.
    URL, which is very similar to how we were doing things before, as it allows you to set an image from the Internet.

In the Quick Links settings, you'll find a brand new customization option as well: icon corner radius. This makes it possible to have perfectly round icons, or perfectly square icons... and everything in between.

Even more link customization!

✨ Settings get a refreshed look

To keep up with 2025's software trend of having everything more round, we've decided to refresh the look of some UI elements in Bonjourr, most notably the settings panel.

We initially were a little on the fence about this new look, but being now used to it, the previous versions of Bonjourr do look noticeably older and less refined, making this refresh very welcome indeed.

⭐ Miscellaneous improvements

In the Weather & Greetings settings, you'll find a new Greeting Mode dropdown that lets you customize how Bonjourr greets you throughout the day: morning, afternoon, evening & night. You can even use the $name variable if you have a name set up.

Bonjourr can now greet you however you want, whenever you want.

Thanks to the help of @ShrekCoding21, you can now choose where the AM/PM indicator appears when using 12h-time mode, allowing for ever more customization.

Additionally, note that some things have been moved around or renamed for clarity:

    Background controls were moved (once again, sorry 😬) to the right-click menu
    Greeting size was moved to advanced options
    Background frequency "Pause" was renamed "Locked"

And lots of technical improvements & bug fixes:

    On Quick Link icons, the perfect ratio between border radius and padding is now respected
    You can now toggle between link groups with your mouse wheel
    When using local files as backgrounds, images under 80Kb won't be compressed anymore
    PNGs under 256px wide/high now use pixelated image rendering (useful for pixel art backgrounds)
    You can now force the native OS context menu by right-clicking + alt key
    Fixed a bug where the backgrounds weren't centered by default
    Touch screen interactions are now more reliable
    Fixed most languages not having a capital letter on the date
    Removed the favicon as emoji input in Edge as Microsoft has broken the functionality (issue #700)
    Fixed an issue in Safari where the JSON settings textarea in "settings management" would be broken when focused on
    Fixed a bug where the settings panel wouldn't be up to date if you used background controls before having opened it
